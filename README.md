# lern-vuejs
lerning vuejs

## install command 

```
nvm use 18.16.1
npm install yarn -g 

npm install -g @vue/cli
# OR
yarn global add @vue/cli

vue create hello-world
cd hello-world/
yarn serve
```


## other command 

```bash
nvm install 18.16.1
nvm install 16.20.2
# OR
nvm uninstall 18.16.1
nvm uninstall 16.20.2
```
